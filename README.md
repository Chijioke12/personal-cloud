
# Personal Cloud (Next.js) - Final Package

This repo contains a ready-to-deploy Next.js personal cloud app with:
- Vercel Blob support (easy integration on Vercel)
- S3 support (AWS or S3-compatible)
- Local filesystem fallback for local dev
- Resumable chunked uploads and client-side AES-GCM encryption demo

Follow setup.md for a step-by-step non-coder deployment guide.
